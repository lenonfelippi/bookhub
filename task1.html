Criei um componente de carrossel de livros utilizando JavaScript, jQuery, HTML5, CSS3 e Bootstrap chamado book-carousel.

Utilizei a base do carrossel do Bootstrap e o adequei ao meu componente.

Para utilizar o componente, você precisa:

  • Ter o Bootstrap (5.3.3) e jQuery (3.5.1) importados na sua página.
  • Importar o script do book-carousel que está no caminho components/book-carousel/book-carousel.js
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous"> <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script> <script src="components/book-carousel/book-carousel.js"></script>

Exemplo:

<book-carousel id="exemplo1" books='[ { "title": "O Senhor dos Anéis: A Sociedade do Anel", "image": "images/o-senhor-dos-aneis-1.jpg", "link": "book-details.html?collection=oSenhorDosAneis&id=1" }, { "title": "O Senhor dos Anéis: As Duas Torres", "image": "images/o-senhor-dos-aneis-2.jpg", "link": "book-details.html?collection=oSenhorDosAneis&id=2" }, { "title": "O Senhor dos Anéis: O Retorno do Rei", "image": "images/o-senhor-dos-aneis-3.jpg", "link": "book-details.html?collection=oSenhorDosAneis&id=3" } ]'></book-carousel>

Note que o componente precisa de 2 atributos: id e books

  • id: Um id único no seu código
  • books: Um array de objetos que contenha title, image e link

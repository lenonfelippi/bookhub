Criei esta página para descrever algumas melhorias que acredito que seriam benéficas para este projeto:

Webpack e SASS

Com o Webpack, conseguiria melhorar um pouco a performance do site, minificando os arquivos. Além disso, com ele, poderia habilitar o .scss ou .less, que gosto muito de usar para criar os estilos do site, criando um CSS mais legível.

Refatoração

Tentaria reduzir os locais onde utilizo as classes do Bootstrap e trocaria pelo meu CSS com BEM, tornando o código mais limpo e fácil de manter.

Cobertura de testes

O ideal seria que todos os componentes tivessem seus testes unitários.

Sugestão de melhoria para o teste

Seria interessante se estivesse disponível uma API para consultar os livros. Com isso, seria possível criar um sistema de busca, lista de livros e página de detalhes mais completa.
